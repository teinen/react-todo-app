# React ToDo App

Create ToDo App to practice React.

## Getting Started

### Installing

Move to your workspace

```
git clone git@github.com:[USERNAME]/react-todo-app.git
```

## Running the application

```
cd react-todo-app
npm start
```

Now application is running, and you can access with:
```
http://localhost:3000
```

## Function
* Create New ToDo
* Complete ToDo
* Delete ToDo
